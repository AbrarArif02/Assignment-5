Question-1: What are the different ways to select an element in the DOM?

Elements in the DOM can be selected using methods like document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector(), and document.querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns or sets the HTML content inside an element, including any nested elements. InnerText returns or sets only the visible text content, ignoring hidden elements. textContent returns or sets the entire text content, including hidden elements but without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in which a parent element handles events for its child elements using event bubbling. This improves efficiency and is useful for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a propagation mechanism where an event starts from the target element and bubbles up to its ancestors, triggering event listeners on each ancestor in the hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Elements can be created using document.createElement(), added with appendChild() or insertBefore(), and removed with removeChild() or remove().
